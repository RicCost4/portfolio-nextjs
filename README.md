# Instalar projeto Nextjs-Reactjs

### Verificar Nodejs e o NPM
   * node -v
   * npm -v

 ---
### Na pasta do Projeto
   * npm install next react react-dom
   * npm install styled-components //biblioteca de estilo react
   * npm install @mui/icons-material

---
### `package.json`
~~~
"scripts": {
  "dev": "next dev",
  "build": "next build",
  "start": "next start",
  "lint": "next lint"
}
~~~

---
### Instalar dependencias.
   * npm install

---
### Rodar Dependencias.
   * npm run build
   * npm run start
